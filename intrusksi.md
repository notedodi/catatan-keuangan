Kamu adalah seorang Senior Fullstack Web Engineer dan UX Designer berpengalaman.

Tugas kamu adalah membuat sebuah WEBSITE CATATAN KEUANGAN PRIBADI
yang sederhana, aman, dan mudah digunakan (mobile-first).

Website ini bersifat PERSONAL (bukan publik), fokus pada pencatatan,
ringkasan, dan analisis keuangan pribadi.

================================================
TUJUAN WEBSITE
================================================
- Mencatat pemasukan dan pengeluaran harian
- Menampilkan ringkasan kondisi keuangan
- Memberikan laporan dan insight sederhana
- Mudah digunakan dalam kehidupan sehari-hari

================================================
FITUR UTAMA & FUNGSINYA
================================================

1. AUTENTIKASI
- Login (password only)
- Proteksi halaman (hanya user login yang bisa akses data)
- Data keuangan bersifat private 

2. DASHBOARD
- Total saldo saat ini
- Total pemasukan bulan ini
- Total pengeluaran bulan ini
- Sisa uang / selisih
- Tombol cepat "Tambah Transaksi"

3. PENCATATAN TRANSAKSI
- Tambah, edit, hapus transaksi
- Field transaksi:
  - Tanggal
  - Jenis (Pemasukan / Pengeluaran)
  - Kategori
  - Nominal
  - Catatan
- Update saldo otomatis

4. KATEGORI KEUANGAN
- Kategori pemasukan & pengeluaran
- Bisa tambah, edit, hapus kategori
- Kategori bisa digunakan untuk analisis

5. RIWAYAT TRANSAKSI
- List semua transaksi
- Filter berdasarkan:
  - Tanggal
  - Bulan / Tahun
  - Kategori
  - Jenis transaksi
- Search transaksi

6. LAPORAN & STATISTIK
- Grafik pemasukan vs pengeluaran
- Grafik pengeluaran per kategori
- Rekap mingguan, bulanan dan tahunan

7. BUDGETING
- Set limit budget per kategori
- Indikator jika mendekati atau melewati limit

8. ANALISIS AI (OPSIONAL)
- Ringkasan otomatis kondisi keuangan bulanan
- Analisis kebiasaan pengeluaran
- Saran penghematan sederhana
- AI hanya membaca data user sendiri

9. EXPORT & BACKUP
- Export data ke PDF / Excel
- Backup data manual

10. PENGATURAN
- Awal bulan laporan
- Tema (light / dark)
- Logout

================================================
USER FLOW
================================================
- User login
- Masuk ke dashboard
- Tambah transaksi harian
- Cek laporan bulanan
- Evaluasi pengeluaran
- Atur budget

================================================
TEKNIS & UI/UX
================================================
- Mobile-first & responsive
- UI sederhana dan bersih
- Form input cepat (minim klik)
- Validasi input yang jelas
- Struktur kode rapi & scalable
